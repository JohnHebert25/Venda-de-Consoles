## Sobre o Projeto

- Este projeto tem como objetivo gerenciar dados relacionados à venda de consoles.
- É uma aplicação focada no uso de banco de dados para registrar, consultar, atualizar e deletar registros de consoles vendidos,
- ideal como trabalho acadêmico ou base para um sistema de controle de estoque/vendas.

## O que contém

- Contém Scrips SQL para criação de dados e inserir dados.
- Create table para criação de tabelas necessárias no banco.
- insert para inserir dados dentro do banco.
- Select para buscar, Update para atualização dentro do banco, Delete para remoção de dados indevidos dentro do banco.

## Tecnologias / Ferramentas utilizadas

- Foi utilizado Banco de dados relacional (SQL)
- Scrip de sql padrão para a criaçao.
